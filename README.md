# Visual Geometry is a package for creating shapes for the UI Toolkit.

> To create shapes, you need to open the window in the tab 'Window/UI Toolkit/Draw Visual Shape', after that you need to click to draw a figure clockwise, then copy the data about the figure. In UI Builder, add a VisualShape from Library to the UXML and paste the data into the Data field.
